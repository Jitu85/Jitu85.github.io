# Jitu's PlayZone Casual Games Portal 🎮

Welcome to **Jitu's PlayZone**, a premium collection of casual and hypercasual browser-based retro games. Jitu's PlayZone runs directly in the browser, offers custom neon aesthetics, PWA capabilities for offline play, search & filtering, and local scoreboard tracking.

## Playable Games
1. **Flappy Rocket**: Navigate your rocket ship through space gates by tapping to thrust.
2. **Stack Tower Builder**: Stack blocks on top of each other, slicing off any overhanging parts.
3. **Reflex Racer**: Switch lanes on a cyber highway to dodge oncoming traffic.

## Features
- **PWA Support**: Installable on desktop/mobile and fully playable offline using Service Worker caching.
- **Dynamic CSS Theme**: Beautifully stylized cyberpunk dark theme with glowing neon cards and fluid transitions.
- **Local Leaderboards**: Compete against pre-populated scores and save your own run records in `localStorage`.
- **Search & Categories**: Filter through the 20-game catalog using tags and instant search matching.
